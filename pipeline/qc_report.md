# 🔍 品質管理部 — QCレポート

動画ごとの品質検査記録。`./pipeline/qc.sh <PaperID>` を実行すると自動で追記される。

## 使い方

```bash
# QC実行（フレーム抽出 + 自動検証）
./pipeline/qc.sh Paper12

# フレームを Finder で確認
open pipeline/qc_frames/Paper12/

# 全論文の data.ts を一括検証
python pipeline/qc_data.py
```

## 目視チェックの観点

| 観点 | 確認内容 |
|------|---------|
| 文字サイズ | 小さすぎて読めないテキストがないか |
| 文字切れ | テキストがフレーム端でトリミングされていないか |
| グラフ | 軸ラベル・数値が正しく表示されているか |
| アニメーション | フェードが滑らかか、震えやジャンプがないか |
| カラー | アクセントカラーが正しく反映されているか |
| Short版 | 縦長フレーム内にコンテンツが収まっているか |
| 音声 | BGMとナレーションのバランス（BGM音量 10%が目安） |
